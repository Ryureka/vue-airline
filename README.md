# 프로젝트 소개
스카이스캐너 API를 이용한 항공권 검색기능과 현지의 가이드가 상품을 올리면  
일반 유저가 상품을 구매할 수 있도록 하는 가이드 서비스 기능이 결합된 웹 어플리케이션을  
Vue.js + Express + MongoDB를 사용하여 구현한 프로젝트입니다.
